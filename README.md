Descripción del desarrollo

**Análisis en Figma**

Se extrajeron tipografías, colores y tamaños directamente del diseño en Figma para mantener la fidelidad visual.
No se utilizaron variables CSS, ya que el caso era sencillo y no requería reutilización compleja.

**Maquetación en HTML**

Se siguieron buenas prácticas semánticas, se replicó el contenido textual proporcionado en el diseño.
